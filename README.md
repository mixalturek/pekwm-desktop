# pekwm-desktop

Configuration files for desktop based on pekwm window manager.


## Installation

- If you are using Debian, you can execute util/install_deb.sh as root to install all necessary packages.
- Manually install software listed in util/install_manual.md file.


## Usage

### Switch keyboard layout

Press both shifts. See bin/reset_keyboard_*.sh scripts (us-cz, us-ru).


### yeahconsole

Press F1 to show/hide the console.
