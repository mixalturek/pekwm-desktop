# pekwm-desktop

Configuration files for desktop based on pekwm window manager.


## Installation

- If you are using Debian, you can execute `util/install_deb.sh` as root to install all necessary packages.
- Manually install software listed in `util/install_manual.md` file.


## Usage

### Window title and buttons

- Window title
    - Left click: Activate
    - Middle click: Close
    - Right click: Show window menu
- Left window button
    - Left click: Always on top
    - Middle click: Shaded (show only the title)
    - Right click: Sticky (visible on all desktops)
- Right window button
    - Left click: Maximize
    - Middle click: Maximize vertical
    - Right click: Maximize horizontal


### Switch keyboard layout

Press both shifts. See `bin/reset_keyboard_*.sh` scripts (us-cz, us-ru).


### yeahconsole

Press F1 to show/hide the console.
